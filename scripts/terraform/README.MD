## Introduction

## Prerequisites

### Azure
To get started, you'll need:
1. install azure cli
2. type 'az login' and log in your azure user.
3. install Terraform cli (https://learn.hashicorp.com/tutorials/terraform/install-cli)
4. run npm start:azure
